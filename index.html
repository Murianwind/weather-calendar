☀️ 기상청 날씨 달력 자동화 제작 가이드

원본 저장소: github.com/Murianwind/weather-calendar

1 프로젝트 포크 (Fork)

상단 원본 저장소 링크 접속 후 오른쪽 상단의 [Fork] 버튼을 눌러 내 계정으로 가져옵니다.

2 기상청 API 인증키 발급

1. 기상청 API 허브 회원가입 후 [마이페이지] → [인증키 현황]에서 키를 복사합니다.

2. 메뉴에서 단기예보, 중기기온, 중기육상예보를 각각 찾아 API활용신청을 완료하세요.

3 GitHub Secrets 설정

내 저장소 Settings → Secrets and variables → Actions에서 KMA_API_KEY라는 이름으로 인증키를 저장합니다.

4 내 동네 설정값 찾기

💡 정확한 검색을 위해 "시/도" 이름을 포함하여 입력해 주세요. (예: 서울 봉화산로, 제주 이호일동)

👇 update_calendar.py에 복사할 값입니다:

  • NX: - / NY: -
  • REG_ID_LAND (육상): -
  • REG_ID_TEMP (기온): -

5 코드 수정 및 자동 생성

1. update_calendar.py 파일을 열고 위에서 찾은 값으로 수정한 뒤 Commit 하세요.

⏳ 자동 빌드 대기
GitHub Actions 탭에서 작업이 완료되면 weather.ics 파일이 생성됩니다.

6 캘린더 구독

생성된 weather.ics 파일의 Raw 주소를 복사하여 스마트폰 달력 앱에 등록하면 끝!
